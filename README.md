## 📄 Selected Research Paper

**Title:** A Data Mining Framework for Building a Web-Page Recommender System

**Domain:** Web-Based Recommender Systems / Data Mining

---

## 🎯 Project Objective

The objective of this project is to study a research-based recommender system and implement its web interface using HTML and CSS. The focus is on replicating the output screens (UI) shown in the research paper.

---

## 🧠 System Overview

The recommender system suggests web pages to users based on their preferences and browsing behavior.

### 🔹 Key Features:

* User Login Interface
* Recommendation Display Page
* Navigation Structure
* User Interaction (Submit/Click actions)

---

## 🖥️ Implemented Pages

1. **Login Page (login.html)**

   * User login form (username & password)
   * Navigation bar (Login, Introduction, DMIS Lab, Help)
   * Create new account button

2. **Recommendation Page (recommendation.html)**

   * Displays recommended web pages
   * Contains clickable links
   * “Submit It!” buttons for interaction
